# Power-BI-Sales-Dashboard
📊 Project Title: Power BI Sales & Distribution Dashboard
Role: Power BI Developer / MIS Analyst

Tech Stack: Power BI Desktop, Power Query (ETL), DAX, GitHub

Project Link: GitHub Repository

📝 Project Overview
The objective of this project was to design and develop an interactive, data-driven dashboard to analyze end-to-end sales performance, distributor trends, and revenue distribution. Raw business data was transformed into professional, actionable visuals to enable stakeholders and management to make fast, informed decisions.

🛠️ Technical Implementation & Key Responsibilities
Data Transformation & ETL: Utilized Power Query to clean, filter, and transform raw data, ensuring a structured data model optimized for performance.

Advanced DAX Formulas: Engineered robust DAX measures using DIVIDE, CALCULATE, and ALL functions to calculate accurate percentage-of-total metrics for dynamic multi-zone analysis.

Dashboard Design & Layout: Developed a clean, corporate-standard 3-column grid layout with integrated multi-button slicers and prominent KPI cards for intuitive navigation and readability.

📈 Core Insights Developed
Executive Overview: Integrated high-level KPI cards to track metrics like Total Revenue, Total Dispatch Value, Total Dispatch Cases, and Active Distributor Count at a single glance.

Zone Performance Analysis: Modeled revenue contributions across multiple zones (GK, NWB1, ROB, NWB2) normalized against a 100% scale to evaluate regional market shares.

Distributor Analysis: Identified the top 5 distributors and evaluated their purchase frequencies alongside their average paying capacities to highlight high-value accounts.

Product Performance: Analyzed the top 10 products and specific SKU-level contributions to understand which products drive the majority of sales revenue.

🎯 Business Impact
This dashboard eliminates the need for manual, time-consuming spreadsheet reporting, reducing report generation efforts by an estimated 30%. It provides the sales and operational teams with an immediate, transparent view of distribution performance and revenue trends.
